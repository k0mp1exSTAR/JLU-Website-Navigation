# JLU-Website-Navigation
因吉林大学相关网站众多，为方便使用吉林大学网络服务，故以 2021 级本科生视角撰写此文。  
研究生相关网址有待完善。  
部分网站具有校区局限性，例：大学物理实验选课网站（南岭）。  
部分网站仅限校内访问或需使用WEBVPN系统。
